Getting Started

Set up a Developer Account

In order to use the ACTIVE.com API you first need to register for a developer account.

After filling out the registration, you will receive a confirmation e-mail with a link where you can register your application to receive a key.

Register Your Application

Next, you will need to register an application.

I know this sounds like you have to have a large-scale, serious application up-front, but all that is required is the name of the application. However, you will probably also want to select your preferred protocol and output format, here we will be using JSON.

Next you will choose which Web APIs you will want to use, we will be choosing the Activity Search API v2. They also have a Campground API, but we won't be covering that in this How-To. You'll also notice that with the Activity Search API we get unlimited calls!

After agreeing to their terms of service, click the Register Application button. We are now presented with our API key, which we will use for our calls.
