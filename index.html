This is a how-to guide that will walk you through setting up, and using, the US Government APIs to access data from across different government agencies.

The US Government APIs allows you access to federal government data in areas ranging from the Federal Aviation Administration(FAA), to the Department of Energy, the USDA and even NASA.

The information you have access to can include recall information, locations of farmer's markets, locations of child safety seat inspection stations or the weather on Mars.

We will only cover a few of the APIs, but if you take a look at the data.gov API catalog you will find over 11,000 datasets you can get access to.

Coming up we'll learn how to:

  • How to get the API key
  • How to make the calls to the API
  • Use the basic call format to allow the users to customize searches
  • Then present the results in a user-friendly manner to give them the details they're looking for
